Cấu hình Obsidian
Để lưu được, cần bật plugin miễn phí "Local REST API" trong Obsidian:
  1. Obsidian → Settings → Community plugins → Browse → cài & bật “Local REST API”.
  2. Mở cài đặt plugin. Dễ nhất: bật công tắc “Enable Non-encrypted (HTTP) Server” (dòng HTTP 27123 sẽ hết dấu ✗ đỏ), rồi chọn HTTP bên dưới.
  3. Copy API Key trong plugin, dán vào ô bên dưới.
  4. Giữ Obsidian mở mỗi khi lưu.
Nếu muốn dùng HTTPS (27124) mà không bật HTTP: phải bấm “this certificate” trong plugin để cài chứng chỉ tin cậy, rồi chọn HTTPS bên dưới.
Không tick ô nào → luôn nối (append) vào cùng một file.
